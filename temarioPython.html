Comience hoy mismo a aprender Python

Los módulos de este lenguaje tipado dinámico, pueden interactuar con una variedad de bases de datos, lo que convierte a python en rey del tema de análisis de datos a gran escala. Este lenguaje es comúnmente utilizado en la fase de aprendizaje introductorio de disciplinas como por ejemplo la ciencia de datos (data science) y el aprendizaje automático (machine learning).

Aprende cómo programar en este lenguaje interpretado con los cursos en línea disponibles en la plataforma de edX. Python te ayudará a desarrollar aplicaciones web y entrar al mundo de la programación, aprenderás conceptos fundamentales de Python.

En los últimos años otros temas relacionados como la Inteligencia Artificial, el deep learning, el machine learning, entre otros han ganado importancia, debido a su importancia. Python también puede ser usado en la ciencia de datos, por ejemplo para analizar big data o gran cantidad de datos. Python es un lenguaje multiplataforma y multiparadigma.
